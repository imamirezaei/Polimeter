# پُلیمتر

آزمون مفهومی فارسی برای سنجش آشنایی با مفاهیم مرتبط با سنت‌های فکری چپ و راست.

## ویژگی‌ها
- کاملا استاتیک و سازگار با GitHub Pages
- بدون AI و بدون هیچ فراخوانی API
- نمره‌دهی محلی و قطعی (deterministic)
- بدون ذخیره پاسخ کاربر
- فقط شمارنده دفعات شرکت (اختیاری) در `localStorage`

## نکته مهم
این آزمون گرایش سیاسی کاربر را تعیین نمی‌کند و فقط میزان آشنایی مفهومی را می‌سنجد.

## اجرای پروژه
### اجرای مستقیم (استاتیک)
فقط `index.html` را باز کنید.

### اجرای لوکال با سرور ساده Node
```bash
npm install
npm run dev
```

پیش‌فرض: `http://localhost:8787`

اگر پورت اشغال باشد، سرور خودکار روی پورت بعدی بالا می‌آید.

## ساختار اصلی
- `index.html`
- `style.css`
- `app.js`
- `data/questions.js`
- `server.js` (اختیاری برای توسعه محلی)

## داده سوال‌ها
هر سوال این فیلدها را دارد:
- `id`
- `type` (`concept | statement | definition`)
- `text`
- `correct_side` (`left | right`)
- `explanation`
- `topic`
- `difficulty` (`easy | medium`)
